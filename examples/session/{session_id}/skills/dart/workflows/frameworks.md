## Visual Aesthetic: 4-Style System for Frameworks

**Recommended styles:** Ligne Claire (primary) or Lithography

### Style Selection

**Ligne Claire (For clear, approachable frameworks):**
- Strong, uniform outlines define framework shape
- Flat, vivid colors in zones/quadrants
- Architectural precision with wit
- Clean, professional appearance

**Lithography (For authoritative frameworks):**
- Fine linework with cross-hatching texture
- Archival, serious aesthetic
- Dense detail suggests depth of thinking
- Limited palette with warm accents

### Core Characteristics
1. **Clear structure** â€” Framework shape is recognizable (2x2, Venn, pyramid, etc.)
2. **Style-appropriate execution** â€” Clean (Ligne Claire) or textured (Lithography)
3. **UL aesthetic** â€” Brand colors, Maastricht Blue linework
4. **Labels integrated** â€” Typography part of visual design
5. **Conceptual clarity** â€” Framework immediately understandable
6. **Memorable visual** â€” Becomes THE reference image for this framework
7. **Thoughtful color** â€” Strategic use of Lawn Green/Tea Green to show relationships
