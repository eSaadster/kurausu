## Purpose

**Use this workflow for:**

- Visible iteration and refinement

- Behind-the-scenes of creation

**Key difference from Recipe Cards:** Recipe Cards are polished step-by-step. Process Documentation shows the ACTUAL messy working process with visible iterations, mistakes, and refinements.

### Step 1: Define the Process Journey

**Identify what you're documenting:**

1. **What process are you showing?**
   - Thinking through a problem
   - Iterating on a design
   - Refining an idea
   - Discovering a solution

2. **What are the key moments?**
   - Initial sketch/attempt
   - Where you got stuck
   - The pivot or insight
   - The refinement
   - The current state (not necessarily "final")

3. **How many iterations?**
   - 3 panels: Quick process (attempt â†’ insight â†’ refinement)
   - 4 panels: Medium process (attempt â†’ struggle â†’ pivot â†’ progress)

### Step 2: Design Panel Progression

**Plan the visual journey:**

1. **Panel evolution:**
   - Each panel shows progression
   - Earlier panels rougher, later panels more refined
   - But ALL panels maintain hand-drawn quality

2. **Visual continuity:**
   - Same subject/concept across panels
   - Visible transformation
   - Elements carry forward (with modifications)

3. **Annotation placement:**
   - Hand-written notes in margins
   - Arrows showing connections
   - Crossed-out text for discarded ideas

**Output:**
```
PANEL LAYOUT: [Horizontal strip / Grid / Staggered]

PANEL PROGRESSION:
Panel 1: ROUGH SKETCH
- Most gestural, loosest lines
- Initial concept visible
- Annotation: "[First thought...]"

Panel 2: STRUGGLE/PROBLEM
- Crossed-out elements
- Question marks, arrows
- Annotation: "[But this doesn't work because...]"

Panel 3: INSIGHT
- Lightbulb moment visible
- New direction emerging
- Annotation: "[Wait, what if...]"
- Crab Red accent on key insight

Panel 4: REFINEMENT
- Building on insight
- More defined but still sketchy
- Annotation: "[So then...]"

Panel 5: CURRENT STATE
- Most refined (but not polished)
- Still shows construction
- Annotation: "[Where I am now...]"

- [What carries through all panels]

```
