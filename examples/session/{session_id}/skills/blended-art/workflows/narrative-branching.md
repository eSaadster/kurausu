## Purpose

Narrative branching visualizes STORY PATHS â€” not technical state diagrams, but narrative journeys with emotional weight. These are **choice trees with soul** â€” showing what happens if you go THIS way vs THAT way.

**Use this workflow for:**
- "Show the story paths"

- Decision consequence visualization
- Alternate timeline illustrations
- "Choose your own adventure" maps

**Key difference from Mermaid:**
- **Mermaid:** Technical state diagrams, precise notation, system logic
- **Narrative Branching:** Storytelling, emotional weight, path consequences

This is about NARRATIVE, not NOTATION.

## Visual Aesthetic: 2-Style Organic System for Narrative Branching

**Recommended styles:** Sumi-e Ink (primary) or Blended Hand-Drawn

### Style Selection

**Sumi-e Ink (PRIMARY â€” Organic narrative flow):**
- Brush strokes flow like rivers of possibility
- Paths branch organically, not geometrically
- Negative space represents unknown futures
- Expressive, emotional branching

**Blended Hand-Drawn (For uncertain/forming narratives):**
- Sketch shows paths still forming
- Some paths clearer than others
- Construction marks show narrative still being written
- Perfect for "what might happen" uncertainty

### Core Characteristics
1. **Organic tree structure** â€” Branches feel like growing, not technical
2. **Story nodes** â€” Each branch point has narrative meaning
3. **Path differentiation** â€” Different paths have different visual weight
4. **Consequence hints** â€” Branches suggest where they lead
5. **Root/origin point** â€” Clear starting point for the story
6. **NOT technical notation** â€” Story flavor, not state diagram
7. **Emotional resonance** â€” Paths have feeling, not just logic

### Step 1: Define the Narrative Structure

**Identify the story you're branching:**

1. **What's the origin point?**
   - Starting situation
   - The moment before choices

2. **What are the major branch points?**
   - Key decisions or forks
   - Moments where paths diverge
   - 2-4 major branch points is ideal

3. **What are the paths?**
   - Where does each branch lead?
   - What's the narrative flavor of each path?
   - Which path is PRIMARY (gets Crab Red)?

4. **What are the endpoints (if any)?**
   - Outcomes, destinations, consequences
   - Or: paths leading into unknown (off-edge)

**Output:**
```
NARRATIVE ORIGIN: [Starting point of the story]

BRANCH POINTS:
1. [First fork]: "[Decision or divergence]"
   - Path A: [Where it leads] â€” [narrative flavor]
   - Path B: [Where it leads] â€” [narrative flavor]

2. [Second fork]: "[Decision or divergence]"
   - Path A leads to: [outcome]
   - Path B leads to: [outcome]

[Continue for all major branches]

PRIMARY PATH (Crab Red): [Which path is highlighted as main/important]

ENDPOINTS:
- [Path leads to]: [Consequence/outcome]
- [Path leads into]: Unknown (off-edge)

NARRATIVE TONE: [Epic/intimate, serious/playful, certain/uncertain]
```

### Step 2: Design the Tree Structure

**Plan the visual narrative:**

1. **Tree orientation:**
   - Top-down (origin at top, outcomes at bottom)
   - Bottom-up (origin at bottom, outcomes rise)
   - Left-to-right (origin at left, time flows right)
   - Radial (origin at center, paths spread outward)

2. **Branch style:**
   - Organic (like a real tree, curving)
   - Rivers (like water branching)
   - Paths (like walking trails forking)
   - Roots (like underground networks)

3. **Node representation:**
   - Small illustrations at decision points
   - Symbolic icons for outcomes
   - Or: pure branch lines with subtle markers

**Output:**
```
TREE ORIENTATION: [Top-down / Bottom-up / Left-right / Radial]

BRANCH STYLE: [Organic tree / Rivers / Paths / Roots]

VISUAL METAPHOR:
- Branches feel like: [Growing things / water / trails / etc.]
- Origin point looks like: [Trunk / source / starting path]
- Branch points show: [Forks / splits / divergences]

NODE REPRESENTATION:
- Decision points: [How they're marked]
- Outcomes: [How they're shown]
- Unknown futures: [How paths trail off]

WEIGHT DISTRIBUTION:
- Primary path: Thickest, Crab Red
- Major alternates: Medium, Gold Leaf
- Minor branches: Thinner, Black
```

### Step 4: Determine Aspect Ratio

| Tree Type | Aspect Ratio | Reasoning |
|-----------|--------------|-----------|
| Vertical tree (origin at bottom) | 9:16 or 3:4 | Tall for upward growth |
| Horizontal flow (left to right) | 16:9 or 21:9 | Wide for temporal flow |
| Radial (origin at center) | 1:1 | Square for symmetry |
| Spreading branches | 16:9 | Wide to show breadth |

**Default: 16:9 (horizontal)** â€” Good for narrative flow left-to-right
