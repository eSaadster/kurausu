### Step 1: Define the Subject

**Gather information:**
```

**Example:**

CONTEXT: OTR driver with 15 years experience, was struggling to find quality positions
SETTING: Standing beside his rig at a truck stop, early morning light
MOOD: Confident, grateful, professional pride
USE CASE: Homepage testimonial section, driver success story blog
```

---

### Step 2: Photography Guidelines

**Technical specifications:**
```
CAMERA SETTINGS:
- Aperture: f/2.8-f/4 (subject isolation)
- Shutter: 1/200+ (sharp subject)
- ISO: As needed for available light
- Format: RAW + JPEG

LIGHTING:
- Natural light preferred (golden hour ideal)
- Avoid harsh midday sun
- Clean catchlight in eyes
- No heavy shadows on face

COMPOSITION:
- Rule of thirds (eyes at upper third)
- Leave space in direction subject faces
- Include environmental context
- Minimum 2/3 body visible
```

**Portrait direction:**
```
EXPRESSION:
- Confident but approachable
- Genuine smile or neutral dignity
- Avoid forced/fake expressions
- Eyes engaged with camera

POSTURE:
- Shoulders back, chest open
- Natural stance, weight distributed
- Hands visible and relaxed
- No crossed arms (closed off)

WARDROBE:
- Work clothes acceptable (clean)
- Company uniform if applicable
- No distracting patterns
- Avoid logos (other than employer)
```

---

### Step 3: Environmental Context

**Setting options:**
```
TRUCK CAB:
- Driver's seat, hands on wheel
- Relaxed but professional
- Dashboard context visible
- Window framing natural light

BESIDE RIG:
- Full or partial truck visible
- Driver as scale reference
- Pride in equipment
- Wide environmental shot

TRUCK YARD:
- Multiple trucks background
- Active but controlled environment
- Fleet context
- Professional setting

LOADING DOCK:
- Industrial environment
- Shows work activity
- Authentic workplace
- Movement and energy

ROAD/HIGHWAY:
- Open road as background
- Freedom, journey themes
- Wide landscape
- American iconography
```

---

### Step 4: Apply Editorial Treatment

**Industrial Editorial style:**
```
COLOR TREATMENT:
- Rich, saturated colors
- Slightly desaturated shadows
- Warm skin tones
- Deep blacks, clean whites

CONTRAST:
- High contrast for impact
- Retain shadow detail
- Bright, clean highlights
- Punchy but not harsh

CROP:
- Full bleed for hero images
- Centered for standalone portraits
- Off-center for layout integration
- Leave room for text overlay if needed
```

**Brand integration:**
```
LIME ACCENT OPTIONS:
- Lime Green #7CFC00 geometric shape overlay
- Lime line/bar element
- Lime glow/highlight effect
- Lime border treatment

TYPOGRAPHY OVERLAY:
- Inter Black for names
- Inter Bold for titles
- Dark Blue #002233 or White (contrast dependent)

COMPOSITION ELEMENTS:
- Angled geometric shapes
- Horizontal lime bars
- Photo edge treatments
- Subtle grid overlay
```

## Driver Representation Guidelines

### Authenticity Principles
```
SHOW:
- Real working conditions
- Genuine expressions
- Professional pride
- Diverse representation
- Age range (21-65)
- Various body types
- Different ethnicities/backgrounds

AVOID:
- Overly staged/fake setups
- Forced smiles
- Stereotypical "trucker" imagery
- Exclusively young/fit models
- Lack of diversity
- Condescending portrayals
```

### Diversity Requirements
```
ACROSS ANY SET OF 4+ PORTRAITS:
- At least 1 woman
- Multiple ethnicities represented
- Age range represented
- Different truck types/settings

INDUSTRY REALITY:
- CDL-A holders: ~70% male, ~30% female (growing)
- Diverse ethnic backgrounds
- Wide age range (25-70)
- Regional variation
```

## Stock Photo Selection

When using stock photography:

### Approved Sources
```
- Getty Images (editorial collections)
- Adobe Stock (authentic selects)
- Stocksy (curated authenticity)
- Industry-specific libraries
```

### Selection Criteria
```
LOOK FOR:
- Natural expressions
- Real work environments
- Appropriate age/diversity
- Quality lighting/composition
- Not obviously staged

AVOID:
- Generic thumbs-up poses
- Perfect studio lighting
- Models who look like actors
- Same photo used everywhere
- Obvious composite images
```

| Mistake | Fix |
|---------|-----|
| Generic stock look | Use editorial-quality, authentic imagery |
| Lack of diversity | Ensure representation across sets |
| Poor lighting | Natural light, golden hour preferred |
| Staged poses | Direct for natural, confident expressions |
| No environmental context | Include truck/industry elements |
| Inconsistent treatment | Apply same color grade across set |
| Missing attribution | Always include name and experience |
| Condescending tone | Treat drivers as professionals they are |
