# Testimonial Cards Workflow

**Creates compelling customer quote visuals that build trust and credibility.**

---

## Purpose

Testimonials are proof. They transform Uptime's claims into verified customer experiences. This workflow creates testimonial visuals that feel authentic, professional, and persuasive.

**Use for:**
- Customer quote sections on landing pages
- Case study pull quotes
- Social proof carousels
- Social media testimonial posts
- Email testimonial graphics
- Sales deck quotes

---

## Recommended Style

**Primary: Industrial Editorial**
Photography-forward with bold editorial treatment for human connection.

**Secondary: Data Dashboard**
Clean, metrics-focused cards for result-oriented testimonials.

### Step 2: Choose Card Type

| Quote Type | Card Type | Best For |
|------------|-----------|----------|
| Short, punchy | Quote Card (Text Only) | Carousels, social proof |
| With customer photo | Photo + Quote | Landing pages, trust building |
| Result-focused | Metric + Quote | ROI sections, data-driven proof |
| Impactful statement | Hero Quote | Page sections, featured testimonials |
| Multiple quotes | Social Proof Grid | Volume/quantity proof |

---

### Step 3: Edit the Quote

**Quote editing guidelines:**
```
DO:
- Keep the authentic voice
- Trim for focus (with [...] if needed)
- Highlight the result or transformation
- Ensure it's specific, not generic

DON'T:
- Change meaning or exaggerate
- Make it sound like marketing copy
- Remove specificity (numbers, timeframes)
- Use without permission
```

**Quote length by format:**
```
Short (< 15 words): Social proof grids, carousels
Medium (15-40 words): Standard cards, featured quotes
Long (40+ words): Hero quotes, case study features
```

## Photo Guidelines

**Technical requirements:**
```
- Minimum: 200 x 200px (for circular crop)
- Ideal: 400 x 400px or higher
- Format: JPG or PNG
- Quality: Professional headshot preferred
- Expression: Confident, approachable
```

**Treatment:**
```
- Circular crop (1:1 ratio)
- Lime Green border (3px)
- No filters or heavy editing
- Consistent lighting across set
```

**Fallback (no photo):**
```
- Use initials in Lime circle
- Or company logo
- Or skip photo entirely (quote-only format)
```

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Generic quote | Use specific results and transformation |
| Too long | Trim to essential message |
| No attribution | Always include name and company |
| Low-quality photo | Use professional headshots only |
| Marketing language | Keep authentic customer voice |
| Missing permission | Get written approval before use |
| Quote marks as text | Use decorative quote marks |
| Inconsistent style | Match all testimonials in a set |
